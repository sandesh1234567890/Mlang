Building MLang: How I Built a Marathi Programming Language from Scratch

Have you ever found yourself thinking in your mother tongue, coding in English, and wishing those two worlds could collide?

It started with a simple chat conversation. I was typing thoughts in Roman Marathi (Marathi words written using the English alphabet)—phrases like "Tujha nav kay?" and "Kai bhava!". That’s when a spark went off. I was studying Compiler Design this semester. I realized that instead of just studying parsers and lexers in textbooks, I could apply these exact concepts to build something personal: a programming language that understands Roman Marathi.

By breaking down language barriers, MLang ensures that every kid in Maharashtra can learn programming at the exact same pace as other students, without being held back by English fluency.

The Concept: Translating Thought to Code

The core idea was simple. Instead of forcing beginners to learn English keywords like if, else, while, and print, they could write code using words they use every day:

  • if becomes jar (जर)
  • else becomes nahitar (नाहीतर)
  • print becomes bol (बोल)
  • input becomes vichar (विचार)
  • while becomes jovar (जोवर)
  • The entry point is chalu (चालू) and the end is bass (बस).

Step-by-Step Implementation

Building a language might sound intimidating, but by breaking it down into modular compiler phases, it became a clear, step-by-step engineering project:

Phase 1 & 2: The Core Transpiler
I decided that the fastest way to run MLang was transpiling—translating MLang code into Python. Python has a rich runtime, so mapping my custom keywords directly to Python code allowed me to run files seamlessly. I created:

  • mlangc (The Compiler): Translates a .m file into a clean, standalone Python .py file.
  • mlang (The Runner): Executes .m files on the fly.

Phase 3: The Lexer
Next, I built a custom Lexer to read source code character-by-character and break it into structural tokens. For example, the statement bol("Hello") is converted into a stream of tokens: BOL, (, STRING, and ).

Phase 4: The Parser & Custom Errors
A parser reads the token stream and verifies if the grammar is correct. Since the compiler was designed for beginners, I wanted error messages that felt friendly and familiar. If someone writes a syntax error, the compiler outputs: Arre! Ithe kahi tari chukla. If they place a token in the wrong place (like jar > 5), the parser catches it and says: He word ithe nako hota / Jar nantar condition pahije.

Phase 5 & 6: Building the Web IDE
To make the experience complete, I built a Web IDE from scratch.

  • Frontend: Built with HTML/CSS and integrated the Monaco Editor (the code editor powering VS Code). I wrote a custom Monarch tokenizer definition to provide real-time syntax highlighting for MLang keywords.
  • Backend: Built a lightweight Python web server that executes the compiler, routes prints, and handles interactive terminal inputs (vichar) in real-time over a request-response protocol.

What I Learned

This project was the perfect final-year project because it bridged the gap between theory and practice:

  • Academic Relevance: I got to implement real-world compiler structures (lexing, AST building, grammar checks) that I learned in my coursework.
  • User Experience: Designing error messages in a native language taught me a lot about making programming accessible and less intimidating.
  • Packaging: Learning how to bundle the tool so it can be installed globally using a simple pip install . on any PC.

MLang started as a simple experiment in a chat window, and now it's a fully-fledged language ecosystem. It goes to show that the best way to learn complex computer science concepts is simply to build something you care about!

🎯 Task 1: Basic Math (साधे गणित)

Write a program that takes two numbers, adds them, and prints the result. 

chalu
a = 15
b = 10
bol(a + b)
bass

🎯 Task 2: Voting Age Check (मतदानाचे वय तपासणे)

Ask the user for their age using vichar, and check if it is greater than or equal to 18. 

chalu
vaya = vichar("Tumche vaya kay? ")
# Translate string input to integer for comparison
vaya_num = int(vaya)

jar vaya_num >= 18
    bol("Eligible!")
nahitar
    bol("Not Eligible!")
bass

🎯 Task 3: Loop Practice (१ ते ५ अंक मोजणे)

Write a loop using jovar (while) to count and print numbers from 1 to 5. 

chalu
i = 1
jovar i <= 5
    bol(i)
    i = i + 1
bass

👨‍🏫 Teacher's Guide (शिक्षकांसाठी मार्गदर्शक)

Use MLang to teach core computer science principles step-by-step:

  • Lexing: Show how the lexer divides Marathi words like jar into language tokens.
  • Parsing: Demonstrate how parser rules detect syntax errors (e.g. typing jar > 5 triggers a custom Marathi error message).
  • Translation: Explain how MLang converts Marathi scripts to Python code before execution.
